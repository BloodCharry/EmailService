# Email Service

Это приложение позволяет создавать email-кампании, отправлять письма подписчикам и отслеживать их открытие.

## Для работы приложения вам понадобятся:

- **Python 2.7**
- **Django 1.9.9**
- **Redis** (для Celery)
- **Docker и Docker Compose** (рекомендуется)
